## 🎮 Gaming & Esports Domain - Full Details

### Why It's Important

- **$200+ Billion Industry** - Larger than movies and music combined
- **3+ Billion Gamers** - Majority of global population
- **Legitimate Skill Development** - Problem-solving, reflexes, strategy, teamwork
- **Social Connection** - Multiplayer, guilds, communities
- **Career Path** - Streaming, esports, content creation

### Trackers

#### 🎮 Gaming Session
```
Fields:
- Game (text): e.g. "Elden Ring", "Valorant"
- Duration (min): 60
- Platform: PC, PlayStation, Xbox, Switch, Mobile, Arcade
- Mode: Casual, Ranked, Competitive, Speedrun, Completionist, Co-op
- Notes: Optional highlights

Example Entry: "🎮 Elden Ring — 90min (Completionist)"
```

#### 🏆 Achievement Hunter
```
Fields:
- Game (text): e.g. "Dark Souls 3"
- Achievement (text): e.g. "Dark Lord"
- Rarity: Common, Uncommon, Rare, Ultra Rare, Legendary
- Type: Trophy, Achievement, Steam, Xbox, PSN, Other

Example Entry: "🏆 Dark Lord trophy in Dark Souls 3"
```

#### 📊 Rank Progress
```
Fields:
- Game (text): e.g. "Valorant", "LoL", "CS:GO"
- Current Rank (text): e.g. "Diamond 2", "Global Elite"
- LP/MMR (optional): e.g. "2500 MMR"
- Peak Rank (optional): e.g. "Ascendant 1"

Example Entry: "📊 Diamond 2 in Valorant (Peak: Ascendant 1)"
```

#### 📹 Streaming/Content
```
Fields:
- Platform: Twitch, YouTube, TikTok, Kick, Other
- Duration (min): 120
- Avg Viewers (optional): 25
- Followers Gained (optional): 5
- Content Type: Live Stream, VOD, Highlight, Tutorial, Other

Example Entry: "📹 Twitch — 120min (25 viewers, +5 followers)"
```
