# Axiom Auto-Progress Logging Feature

## Overview

Axiom now automatically logs user progress to the notebook during nightly operations. When the midnight scan detects significant progress changes (≥10%), it creates detailed notebook entries documenting the progress updates.

## Features

### 1. Individual Progress Updates

For each goal with significant progress change (≥10%), Axiom creates a notebook entry with:

- **Progress Change**: Before → After percentages
- **Status Badge**: Based on new progress level
  - 🎉 COMPLETED (100%)
  - 🔥 Almost There (≥75%)
  - ✅ Halfway (≥50%)
  - 🚀 Making Progress (≥25%)
  - 🌱 Getting Started (<25%)
- **AI Analysis**: Reasoning behind the progress update
- **Metadata**: Tagged for filtering and tracking

### 2. Daily Progress Summary

At the end of each nightly scan, Axiom creates a summary entry with:

- **Date**: Formatted date header
- **Statistics**:
  - Total Goals Updated
  - Progress Increases
  - Goals Completed
- **Progress Details**: List of all changes with emoji indicators
- **Auto-generated Tag**: For easy identification

### Entry Structure

#### Individual Progress Entry

```typescript
{
  entry_type: 'goal_progress',
  title: '📈 Progress: [Goal Name]...',
  content: `## Axiom Progress Update 📈

**Progress Change:** 45% → 65% (+20%)
**Status:** 🚀 Making Progress

### Analysis
[AI reasoning here]

---
*This progress was automatically logged by Axiom during nightly analysis.*`,
  domain: 'personal_development',
  source_table: 'goal_trees',
  source_id: '[goal-id]',
  metadata: {
    auto_logged: true,
    old_progress: 45,
    new_progress: 65,
    change: 20,
    logged_by: 'axiom_midnight_scan',
  }
}
```

#### Daily Summary Entry

```typescript
{
  entry_type: 'note',
  title: '🎉 Daily Progress Summary - [Date]',
  content: `## 🎉 Daily Progress Summary - Monday, January 15, 2026

**Total Goals Updated:** 3
**Progress Increases:** 3
**Goals Completed:** 1

### Progress Details

📈 **Goal 1**: 45% → 65% (+20%)
📈 **Goal 2**: 80% → 95% (+15%)
🎉 **Goal 3**: 90% → 100% (+10%)

---
*Summary automatically generated by Axiom during nightly analysis.*`,
  domain: 'personal_development',
  metadata: {
    auto_logged: true,
    summary_type: 'daily_progress',
    total_updates: 3,
    positive_updates: 3,
    completed_goals: 1,
    logged_by: 'axiom_midnight_scan',
    date: '2026-01-15',
  }
}
```

### Thresholds

- **Individual Logging**: Only logs changes ≥10%
- **All Updates Summarized**: Daily summary includes all changes regardless of size
- **Significant Change Detection**: Progress updates only applied if change >5%

## User Experience

### What Users See

When users open their notebook in the morning, they'll find:

1. **Individual Progress Entries** (for significant changes)
   - Detailed analysis of each major progress update
   - AI-generated insights about their progress
   - Clear visual indicators (emoji, status badges)

2. **Daily Summary Entry**
   - Quick overview of all progress made
   - Celebration of completed goals
   - Motivation from seeing cumulative progress

## Technical Notes

### Performance

- Only processes active users (logged activity in last 7 days)
- Batches all progress updates efficiently
- Minimal database overhead (uses existing progress estimation data)

### Filtering

Users can filter auto-logged entries using:

- `metadata.auto_logged = true`
- `entry_type = 'goal_progress'`
- `metadata.logged_by = 'axiom_midnight_scan'`

### Customization

Thresholds can be adjusted in code:

```typescript
// Current threshold: 10%
if (Math.abs(change) < 10) {
  continue; // Skip small changes
}
```
