 2026-04-21 — AI Agent integration (OpenClaw, Hermes, Lindy, etc.)
 --------------------------------------------------------------------

 1. Run the new migration:

    psql -h db.xxx.supabase.co -U postgres -d postgres -f migrations/2026-04-21-ai-agents.sql

 2. This creates:
    - public.agents (predefined agent whitelist)
    - public.agent_keys (user-authorized API keys)
    - public.agent_connect_codes (temp OAuth codes)

 3. No env changes required.

 Usage:
    - Users visit /settings → "Connect AI Agent" → click agent to authorize
    - Agent redirects with code → calls /api/agent/keys/exchange → gets api_key
    - Agent uses X-API-Key header in requests

 Users can revoke access anytime via /api/agent/keys.
