## 📊 Telegram Best Practices

### Channel vs Group

| Tipo        | Scopo                          | Frequenza                |
| ----------- | ------------------------------ | ------------------------ |
| **Channel** | Broadcast (solo admin postano) | 1-2 post/giorno          |
| **Group**   | Community (tutti postano)      | Monitorare costantemente |

### Formattazione

**Usa Markdown:**

```
**Grassetto** per enfasi
__Corsivo__ per citazioni
`Monospace` per codice
[Link](url) per CTA
```

**Emoji strategiche:**

- 🚀 per launch/annunci
- 💡 per tips
- 🎉 per celebrazioni
- ⚠️ per alert
- 👉 per CTA

### Orari Migliori

| Tipo              | Orario Migliore                 |
| ----------------- | ------------------------------- |
| Channel broadcast | 9:00, 13:00, 20:00 CET          |
| Group engagement  | 12:00-14:00, 19:00-21:00 CET    |
| Direct messages   | 10:00-17:00 CET (orario lavoro) |

### Moderation Group

**Regole da fissare:**

1. No spam o self-promotion
2. Rispetto per tutti i membri
3. No contenuti offensivi
4. Stay on topic (accountability, produttività)
5. Usa thread per domande specifiche

**Bot consigliati:**

- @GroupHelpBot (moderation)
- @LikeBot (reazioni)
- @PollBot (sondaggi)
