## 📊 Pinterest Best Practices

### Formato Pin

| Elemento   | Specifica                         |
| ---------- | --------------------------------- |
| Dimensioni | 1000x1500px (2:3 ratio)           |
| Font size  | Minimo 24pt (leggibile da mobile) |
| Colori     | Contrasto alto                    |
| Branding   | Logo Praxis in angolo             |
| Testo      | Max 30% dello spazio              |

### SEO Pinterest

**Titolo Pin:**

- Max 100 caratteri
- Keyword principale all'inizio
- Esempio: "Productivity Tips: Come Smettere di Procrastinare"

**Description:**

- Max 500 caratteri
- 2-3 keyword naturali
- Call-to-action chiara
- Esempio: "Scopri come costruire abitudini solide con Praxis..."

**Hashtag:**

- 2-5 hashtag rilevanti
- Mix di grandi (#Productivity) e nicchia (#PraxisApp)
