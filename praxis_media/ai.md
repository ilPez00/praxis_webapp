```

4. 3 keyword transazionali (pronte per acquisto)
5. 3 keyword informative (top of funnel)

- Priorità: bassa difficoltà (<30), volume decente (>100)
- Focus: long-tail per ranking veloce
- Content gap: cosa fanno competitor che noi no
```

## Prompt per Analisi Dati

```
Sei un data analyst per SaaS.

Analizza cohort retention per Praxis.

Dati disponibili:
- Utenti signup per settimana
- Check-in per utente per giorno
- Upgrade a Pro per utente

1. Calcola retention per coorte (settimana di signup)
   - Giorno 1, 3, 7, 14, 30
2. Identifica pattern:
   - Quali coorti hanno retention migliore?
   - Cosa hanno fatto di diverso?
3. Correlazioni:
   - Utenti con partner hanno retention migliore?
   - Utenti che fanno check-in mattina vs sera?
   - Utenti che completano onboarding vs no?
4. Raccomandazioni:
   - 3 azioni per migliorare retention
   - Priorità (impatto / sforzo)

Output:
- Tabella retention per coorte
- Grafico suggerito (line chart)
- Insight chiave (3 bullet)
- Azioni raccomandate (3 bullet)
```

### 24. Analisi LTV:CAC

```
Sei un data analyst per SaaS.

Calcola e analizza LTV:CAC per Praxis.

Dati disponibili:
- MRR per utente
- Churn rate mensile
- CAC per canale (organico, Google Ads, Meta Ads, etc.)
- Costo operativo per utente

Task:
1. Calcola LTV per utente medio
   Formula: MRR / Churn Rate × Margine Lordo
2. Calcola CAC per canale
3. Calcola LTV:CAC per canale
4. Identifica:
   - Canale con LTV:CAC migliore
   - Canale con LTV:CAC peggiore
   - Canale da scalare
   - Canale da tagliare
5. Raccomandazioni:
   - Budget allocation per canale
   - Target LTV:CAC (3:1 è sano)
   - Come migliorare LTV (retention, upsell)
   - Come ridurre CAC (ottimizzazione conversione)

Output:
- Tabella LTV:CAC per canale
- Grafico suggerito (bar chart)
- Raccomandazioni budget
- 3 azioni per migliorare ratio
```

## 🎯 Come Usare Questi Prompt

### Best Practices

1. **Personalizza sempre**
   - Sostituisci [VARIABILI] con dati reali
   - Aggiungi contesto specifico di Praxis
   - Adatta tono al canale

2. **Itera sui risultati**
   - Primo output è una bozza
   - Chiedi revisioni ("rendilo più conciso", "aggiungi esempi")
   - A/B test copy diversi

3. **Combina prompt**
   - Es: Blog post → Thread Twitter → Post LinkedIn → Carousel Instagram
   - Un contenuto, 10 formati diversi

4. **Salva output migliori**
   - Crea swipe file di copy che funzionano
   - Usa come riferimento per prompt futuri
