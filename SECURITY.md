# Security Policy

## Supported Versions

Currently, the latest version of Praxis is supported for security updates.

| Version | Supported |
| ------- | --------- |
| 1.0.x   | ✅         |

## Reporting a Vulnerability

We take the security of Praxis seriously. If you believe you have found a security vulnerability in this project, please do NOT report it via the public issue tracker.

Instead, please report it privately to the maintainers. You can reach out via [INSERT CONTACT METHOD].

Please include the following information in your report:

- Type of issue (e.g., SQL injection, cross-site scripting, etc.)
- Full paths of source file(s) related to the manifestation of the issue
- A proof of concept or instructions to reproduce the issue
- Possible impact of the vulnerability

We will acknowledge your report within 48 hours and provide a timeline for resolution.
