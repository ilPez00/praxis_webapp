```

  → POST /api/notebook/capture { imageUrl, text? }
  → Backend: AxiomMultimodalService base64-encodes image
  → Backend: AICoachingService.runMultimodal() with Gemini vision (gemini-2.0-flash+)

  → Backend: Returns { analysis, domain, tags, notebookEntry, diaryEntry }
  → Frontend: Shows Axiom's analysis overlay (domain chip, tag chips, insight text)
  → User can add text note → "Saved to Notebook" → navigates to /notes
```
