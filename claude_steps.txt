# Session: March 11, 2026 - Axiom Management and Key Strategy

## Summary of Changes
- **Axiom Key Strategy (Backend/Frontend):**
  - **Dynamic Strategy:** Updated `AICoachingService.ts` to fetch the `axiom_key_strategy` from the `system_config` table. Axiom now starts its key iteration based on this preference (First, Last, or Random).
  - **Admin UI Controls:** Added a new **"API Key Pool Strategy"** section to the Admin > Axiom tab. Admins can now toggle between three modes:
    - **Start from First:** Always starts with the first key in the `.env` list.
    - **Start from Last:** Always starts with the last key in the `.env` list.
    - **Balanced (Random):** Distributes load evenly (original default).
  - **Instant Application:** Any change made in the admin panel is applied to all subsequent Axiom requests.
- **Diagnostics & Persistence:**
  - Improved the Admin UI to refresh all Axiom settings (prompt and strategy) every time the tab is opened.
